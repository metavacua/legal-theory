# Court Record

Court-record documents in this repository are split into two kinds:

- **[matters/](matters/)** — 5 active matters, each with a complaint-structure `README.md`,
  a `findings.md` (FRCP Rule 52 findings-of-fact/conclusions-of-law stub), and an `evidence/`
  corpus of supporting research.
- **[theory/](theory/)** — legal-authority research organized by jurisdictional level
  (federal constitutional, California constitutional, federal statutes, California statutes,
  municipal), each split into existing doctrine/law, extensions, specializations, and reversal
  arguments.

For a complete, titled, file-level listing of every document in both, see
**[docs/index.md](../index.md)** — the [Matters](../index.md#matters) and
[Theory](../index.md#theory) sections.
